## Combat flow canon

1. Battle start : Archangel scripted Moon That Never Sets submap 729
2. Pattern : ~Sword Bash / Spark Net / Flash Hall / Trans Light / Spectral Flash actions standard
3. À HP < 75% : Talk #1 + Blow Trumpet single-use
4. À HP < 60% : Talk #2
5. À HP < 35% : Talk #3 + Blow Trumpet 2nd single-use
6. **À HP ≤ 31% Meru** : Healing Flower OR Healing Scripture (heal Meru)
7. À HP < 5% : Talk #4 → **Final Verdict → battle ends** victory condition
