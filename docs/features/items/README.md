## Vue d'ensemble système canon

- **5 slots équipement par membre** : Weapon + Headwear + Body Armor + Footwear + Accessory
- **Pas de changement en combat** : équipement figé pendant battle (system screen seulement)
- **Inventaire max 255** items (gameplay constraint)
- **Restrictions canon** :
  - **Weapons** : 100% character-locked (Dart=swords seulement, Lavitz/Albert=spears, Shana/Miranda=bows, Rose=daggers/rapiers, Haschel=fists, Meru=hammers/maces, Kongol=axes)
  - **Headwear** : split Men/Women + 5 par persona spécifique + 5 "All"
  - **Body Armor** : split Men/Women + character-spécifique pour Haschel/Kongol + 7 Dragoon (DG) armors elemental immunity
  - **Footwear** : Men/Women split + 1 "All" (Magical Greaves)
  - **Accessories** : majoritairement "All" + 4 restricted (Bandit's Ring men, Dancer's Ring women, Wargod Calling/Ultimate Wargod not Shana/Miranda)
