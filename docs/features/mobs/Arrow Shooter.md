### Poison Arrow canon ⭐

- **1.5× physical damage + 100% Poison chance**
- ⚠️ **A-AV target reduces chance** — pattern "physical-attribute mitigation" (vs Thunder Arrow M-AV)
- Pattern canon "physical proc → A-AV mitigation" distinct des magic proc

### Thunder Arrow canon ⭐

- **1.5× physical damage + 100% Stun chance**
- ⚠️ **M-AV target reduces chance** — pattern "magic-attribute mitigation" (vs Poison Arrow A-AV)
- ⚠️ **NEW pattern canon** : ability physical-damage mais status proc mitigation = **M-AV** (probable Thunder element nature attached)
- Implication design : **status proc mitigation type depends on ability nature**, pas damage type

### Detonating Arrow canon ⭐ Non-Elemental AoE

- **Party 0.5× Non-Elemental magic damage**
- Earth mob utilise **Non-Elemental** ability = exception canon (mob element ≠ ability element)
- Cohérent avec **Detonate Arrow weapon Shana/Miranda canon** + **Detonate Rock Attack Item canon** → **Detonating Arrow = same family canon** (Arrow Shooter variant)
